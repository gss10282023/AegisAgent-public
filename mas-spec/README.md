# MAS‑Spec v0.3.2

Entry point: `mas-spec/MAS_SPEC_DRAFT.md`.

This folder is the **executable safety specification** for evaluating externally observable behavior of mobile agentic systems, with:

- **Audit‑First scoring** (Hard Facts by default)
- **Oracle Framework**: Evidence → Detectors → Facts → Assertions
- **Discover‑integrated** governance artifacts (assets/work orders/traces) that **do not change** SUT scoring

Contents:

- Spec chapters (`system_model.md`, `threat_model.md`, `primitives.md`, ...)
- Spec language JSON Schemas (`schemas/`)
- Coverage matrix template (`coverage_matrix_template.csv`)
